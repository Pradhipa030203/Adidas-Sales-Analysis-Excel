This project analyzes Adidas sales data using Microsoft Excel to identify business trends and insights.

- Charts & Graphs
- Data Cleaning

- Profit insights

- KPI metrics

## Key Insights

- High revenue products
- Seasonal sales trends

This project demonstrates data analysis and visualization skills using Excel for business decision-making.
